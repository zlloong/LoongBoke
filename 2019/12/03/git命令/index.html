git配置

建立一个文件夹
git init 初始化在当前文件夹
git remote add origin 仓库链接 打开
git clone 厂库链接 克隆
git add 添加当前本地
git status 查看有无更更改
git commit -m “修改元素” 提交更改
git push origin master 上传git
