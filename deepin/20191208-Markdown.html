Markdown

Posted on 2019-12-08 Edited on 2019-12-18 In deepin

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

基础语法格式

标题

1.标题 #号 几个代表几级标题
2.水平线 三个* _ - 都可以

文本样式

1.加粗 俩个* _
2.斜线 一个_ *
3.粗斜体 三个
3.文字带线(删除线) 俩个~
4.带下划线可以用 html变迁<\u>

列表

无序 + * -
有序 1. 2. 3.
嵌套 加tap 或者四个空格
###区块引用
引用是 >加空格
嵌套是多家 >加空格嵌套
> > > 第二层嵌套
列表加区块 加tap 或者四个空格

代码

三个包裹 可加名称 \`` more
或者 tap 四个空格 退格

高级

链接

[链接名称](链接地址)或者<链接地址>
链接也可以用变量来代替，俩个方括号 最后调用

图片

![alt 属性文本](图片地址 “可选标题”)
开头一个感叹号 !
接着一个方括号，里面放上图片的替代文字
接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字
也可以用变量

表格

| 表头 | 表头 |
| —- | —- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |
Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。
表头设置
-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐。

其他

不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。
